【Function 指令說明】
1. 範例： 
const axios = require('axios');
const res = await axios.get('https://api.github.com');
return res.data;
2. msg 和 client 參數可用，例如 msg.reply() 或 client.sendMessage(...)。
3. 如需第三方套件，請先於主機 npm install，並用 require 載入。
4. 不能使用 import，只能 require。
5. 你寫的內容會自動包裝成 module.exports = async function(msg, client) { ... } 並儲存於 data/functions/指令ID.js。
